# Project Operations and Field Service integration overview

The integration between Dynamics 365 Project Operations and Field Service connects field service execution with project and financial management. Instead of managing field service execution and project accounting in separate systems, the integration connects work orders, projects, project tasks, labor, materials, time entries, and financial transactions. Service activities performed in the field flow into Project Operations, and when enabled, Dynamics 365 Finance.

This approach reduces manual reconciliation, improves visibility across service and project activities, and supports more predictable financial and operational outcomes.

Learn more in [Field Service integration with Project Operations overview](/dynamics365/field-service/project-operations-integration).

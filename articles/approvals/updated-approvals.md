# Navigating Approvals

[!INCLUDE[banner](../includes/banner.md)]
[!INCLUDE[banner](../includes/preview-note.md)]

_**Applies To:** Project Operations Integrated with ERP, Project Operations Core_

The approvals experience Microsoft Dynamics 365 Project Operations has been updated as of version 4.0.164.x. By turning on the "Time-phased approvals" feature in the feature control menu. This hosts all relevant approvals views inside a form on the new *Approvals* entity (msdyn_projectapprovalheader).
